🔄 변경된 게임 방식
이전
상단: 영어 단어 (acquire)
블록: 한국어 뜻들 (정답=초록, 오답=빨강)
변경 후
┌─────────────────────────────┐
│   이 뜻의 영어 단어를 맞춰라!  │
│         "획득하다"           │  ← 한국어 뜻
│           100점              │
└─────────────────────────────┘

    ┌──────────┐ ┌──────────┐
    │ acquire  │ │ adequate │  ← 모두 같은 색!
    └──────────┘ └──────────┘
    ┌──────────┐ ┌──────────┐
    │ advocate │ │ allocate │  ← 정답 힌트 없음
    └──────────┘ └──────────┘
✨ 주요 변경점
항목	내용
상단 표시	한국어 뜻 (예: "획득하다")
블록 표시	영어 단어 4개
블록 색상	모두 동일 (인디고 블루)
정답 힌트	❌ 없음
정답 시	🟢 초록 폭발 + "정답! +100"
오답 시	🔴 빨간 폭발 + "오답! -20점"
